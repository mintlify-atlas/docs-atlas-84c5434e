---
title: Update Group
description: Update an existing bookmark group
---

## Endpoint

```
group.update
```

Updates the name or color of an existing group.

## Authentication

Requires authentication via session cookie. Users can only update groups they own.

## Request

<ParamField name="id" type="string" required>
  ID of the group to update
</ParamField>

<ParamField name="name" type="string">
  New name for the group
</ParamField>

<ParamField name="color" type="string">
  New hex color code for the group
</ParamField>

## Response

Returns the updated group object.

<ResponseField name="id" type="string" required>
  Unique identifier for the group
</ResponseField>

<ResponseField name="name" type="string" required>
  Updated group name
</ResponseField>

<ResponseField name="color" type="string" required>
  Updated hex color code
</ResponseField>

<ResponseField name="userId" type="string" required>
  ID of the user who owns the group
</ResponseField>

<ResponseField name="createdAt" type="date" required>
  Timestamp when the group was created
</ResponseField>

<ResponseField name="updatedAt" type="date" required>
  Timestamp when the group was last updated
</ResponseField>

## Example

```typescript
import { client } from '@/lib/client';

const group = await client.group.update({
  id: "grp_123",
  name: "Development",
  color: "#10b981"
});

console.log(group);
// {
//   id: "grp_123",
//   name: "Development",
//   color: "#10b981",
//   userId: "usr_456",
//   createdAt: Date,
//   updatedAt: Date
// }
```
